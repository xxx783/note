## 🎯 核心功能

### 1. 动画系统 (MiuixAnimation.kt)

**缓动函数库**：
- ✅ `SpringEasing` - 弹簧缓动
- ✅ `BounceEasing` - 弹跳缓动
- ✅ `CubicBezierEasing` - 三次贝塞尔缓动
- ✅ `MiuixEasings` - 缓动工具类（包含 Spring、Bounce、Accelerate、Decelerate、Linear）

**动画修饰符**：
- ✅ `miuixSpring()` - 透明度弹簧动画
- ✅ `miuixScaleSpring()` - 缩放弹簧动画
- ✅ `miuixRotateSpring()` - 旋转弹簧动画
- ✅ `miuixTranslateSpring()` - 平移弹簧动画

**状态动画**：
- ✅ `rememberMiuixState()` - 状态容器
- ✅ `MiuixState` - 状态管理类

**交互动画**：
- ✅ `MiuixClickableBox()` - 可点击动画容器
- ✅ `MiuixAnimatedListItem()` - 列表项入场动画
- ✅ `MiuixPulseAnimation()` - 脉冲动画

### 2. UI 组件 (MiuixComponents.kt)

**基础组件**：
- ✅ `MiuixButton` - MIUI 风格按钮（带弹簧动画）
- ✅ `MiuixCard` - MIUI 风格卡片
- ✅ `MiuixTextField` - 带清除动画的输入框
- ✅ `MiuixSwitch` - 弹簧动画开关
- ✅ `MiuixSlider` - 平滑动画滑块
- ✅ `MiuixProgressBar` - 进度条
- ✅ `MiuixBadge` - 徽章组件
- ✅ `MiuixDialog` - 带入场动画的对话框

## 🚀 如何使用

### 方法一：集成到现有项目

```gradle
// settings.gradle
include ':miuix-compose'
project(':miuix-compose').projectDir = file('miuix-compose')

// app/build.gradle
dependencies {
    implementation project(':miuix-compose')
    implementation platform('androidx.compose:compose-bom:2024.06.00')
}
```

### 方法二：使用 AAR 文件

```bash
# 构建 AAR
cd miuix-compose
./gradlew assembleRelease

# 复制 AAR 到您的项目
cp build/outputs/aar/miuix-compose-release.aar /path/to/your/app/libs/
```

### 方法三：发布到 JitPack


## 📝 代码示例

### 基础动画

```kotlin
@Composable
fun SimpleAnimation() {
    var expanded by remember { mutableStateOf(false) }
    
    val scale by animateFloatAsState(
        targetValue = if (expanded) 1.2f else 1f,
        animationSpec = spring(dampingRatio = 0.75f, stiffness = 300f)
    )
    
    MiuixButton(
        onClick = { expanded = !expanded },
        text = "点我"
    )
    
    Box(
        modifier = Modifier
            .size(100.dp)
            .scale(scale)
            .background(Color.Blue)
    )
}
```

### 使用组件

```kotlin
@Composable
fun ComponentsDemo() {
    var text by remember { mutableStateOf("") }
    var checked by remember { mutableStateOf(false) }
    
    Column {
        MiuixTextField(
            value = text,
            onValueChange = { text = it },
            placeholder = "请输入"
        )
        
        Row(verticalAlignment = Alignment.CenterVertically) {
            Text("开关")
            MiuixSwitch(
                checked = checked,
                onCheckedChange = { checked = it }
            )
        }
        
        MiuixButton(
            onClick = { /* 处理 */ },
            text = "提交"
        )
    }
}
```

### 列表动画

```kotlin
@Composable
fun AnimatedList(items: List<String>) {
    LazyColumn {
        itemsIndexed(items) { index, item ->
            MiuixAnimatedListItem(index = index) {
                Card {
                    Text(item, modifier = Modifier.padding(16.dp))
                }
            }
        }
    }
}
```
