## 🚀 如何使用

### 1. 发布到 JitPack

```bash
# 1. 将代码推送到 GitHub
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/yourusername/miuix.git
git push -u origin main

# 2. 创建 Release Tag
git tag v1.0.0
git push origin v1.0.0

# 3. 在 GitHub 创建 Release
# 访问 https://github.com/yourusername/miuix/releases
```

### 2. 用户安装方式

```gradle
// 项目级 build.gradle
allprojects {
    repositories {
        maven { url 'https://jitpack.io' }
    }
}

// 模块级 build.gradle
dependencies {
    implementation 'com.github.yourusername.miuix:core:1.0.0'
    implementation 'com.github.yourusername.miuix:animation:1.0.0'
    implementation 'com.github.yourusername.miuix:widget:1.0.0'
}
```

### 3. 使用示例

```kotlin
// Kotlin 示例
import miuix.animation.Folme
import miuix.animation.easing.SpringEasing

val view: View = findViewById(R.id.my_view)

// 弹簧动画
Folme.useValueTarget(view)
    .addTarget("alpha", 0f, 1f)
    .setEasing(SpringEasing())
    .start()
```

```java
// Java 示例
import miuix.animation.Folme;

View view = findViewById(R.id.my_view);

Folme.useValueTarget(view)
    .addTarget("alpha", 0f, 1f)
    .start();
```

## 📊 模块说明

### core (核心模块)
- **包名**: `miuix.core`
- **功能**: 基础工具类、系统属性读取
- **依赖**: 无
- **大小**: ~50 KB

### animation (动画模块)
- **包名**: `miuix.animation`
- **功能**: Folme 动画引擎、物理动画、缓动函数
- **依赖**: core
- **大小**: ~500 KB
- **核心类**: 
  - `Folme` - 动画主入口
  - `SpringAnimation` - 弹簧动画
  - `SpringEasing` - 弹簧缓动

### widget (控件模块)
- **包名**: `miuix.widget`
- **功能**: MIUI 风格自定义控件
- **依赖**: core
- **大小**: ~100 KB
- **核心类**:
  - `ClearableEditText` - 可清除文本框
  - `SeekingBar` - 自定义拖动条
